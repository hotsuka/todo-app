# TODO App

バニラJavaScript（HTML/CSS/JS）で作成されたフル機能のTODOアプリケーションです。

## 機能

### 基本機能
- ✅ **CRUD操作**: TODOの追加、編集、削除、完了マーク
- ✅ **優先度設定**: 高・中・低の3段階
- ✅ **期限設定**: 日付ピッカーで期限を設定
- ✅ **カテゴリ**: 複数のカテゴリを設定可能
- ✅ **タグ**: ハッシュタグでTODOを整理

### フィルタ・検索
- 🔍 **テキスト検索**: タイトルと説明を検索
- 📊 **ステータスフィルタ**: 全て/未完了/完了済み
- 🎯 **優先度フィルタ**: 優先度別にフィルタ
- 🏷️ **カテゴリフィルタ**: カテゴリ別にフィルタ

### データ管理
- 💾 **自動保存**: LocalStorageに自動保存
- 📤 **エクスポート**: JSONファイルでバックアップ
- 📥 **インポート**: JSONファイルから復元

### UI/UX
- 📱 **レスポンシブデザイン**: モバイル対応
- 🎨 **モダンなデザイン**: 直感的なインターフェース
- ⚡ **高速**: バニラJSによる軽量実装

## 使い方

### 起動方法

1. **ブラウザで開く**
   ```
   C:\Users\the-b\todo\index.html
   ```
   をブラウザで開いてください。

2. **ローカルサーバーで実行（推奨）**
   ```bash
   # Pythonがインストールされている場合
   python -m http.server 8000

   # Node.jsがインストールされている場合
   npx http-server
   ```
   その後、`http://localhost:8000` にアクセスします。

### 基本操作

#### TODOの追加
1. フォームにタイトルを入力（必須）
2. 必要に応じて説明、優先度、期限、カテゴリ、タグを設定
3. 「追加」ボタンをクリック

#### TODOの編集
1. TODOアイテムの「✏️」ボタンをクリック
2. フォームに内容が表示されるので編集
3. 「更新」ボタンをクリック

#### TODOの完了
- チェックボックスをクリックして完了/未完了を切り替え

#### TODOの削除
- 「🗑️」ボタンをクリックして削除（確認ダイアログが表示されます）

#### フィルタ・検索
- **テキスト検索**: 検索ボックスにキーワードを入力
- **ステータス**: 「全て」「未完了」「完了済み」ボタンで切り替え
- **優先度**: ドロップダウンから選択
- **カテゴリ**: ドロップダウンから選択

### データ管理

#### エクスポート
1. 画面下部の「エクスポート」ボタンをクリック
2. JSONファイルがダウンロードされます

#### インポート
1. 画面下部の「インポート」ボタンをクリック
2. 以前エクスポートしたJSONファイルを選択

#### 全削除
1. 画面下部の「全削除」ボタンをクリック
2. 確認ダイアログで「OK」を選択

## 技術仕様

### 技術スタック
- **HTML5**: セマンティックマークアップ
- **CSS3**: CSS変数、Flexbox、Grid
- **JavaScript (ES6+)**: モジュール、クラス

### アーキテクチャ
- **Model**: `Todo.js`, `TodoList.js`
- **View**: `TodoForm.js`, `TodoItem.js`, `FilterBar.js`, `TodoStats.js`
- **Service**: `StorageService.js`
- **Controller**: `app.js`

### ブラウザ要件
- LocalStorage対応ブラウザ
- ES6+ モジュール対応ブラウザ
- 推奨: Chrome, Firefox, Safari, Edge (最新版)
